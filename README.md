# RusChords

**RusChords** — это веб-приложение для гитаристов, объединяющее библиотеку аккордов русских песен, тюнер и метроном в одном удобном интерфейсе.

 **Live Demo:** [https://blackvlex.github.io/guitarapp/](https://blackvlex.github.io/guitarapp/)

---

## Возможности

-  **Каталог песен:** Коллекция из 15+ популярных русских песен (Цой, ДДТ, Кино, Любэ и др.) с готовыми аккордами.
-  **Умный поиск:** Мгновенный поиск песен по названию или исполнителю.
-  **Фильтры:** Сортировка по сложности (Легко, Средне, Сложно) и по исполнителю.
-  **Транспозиция:** Возможность менять тональность аккордов в один клик.
-  **Метроном:** Встроенный метроном с выбором BPM (40-200) и размера такта (4/4, 3/4, 6/8).
-  **Тюнер:** Точная настройка гитары через микрофон с использованием Web Audio API.
-  **PWA (Offline):** Приложение работает без интернета и может быть установлено на телефон как нативное.
-  **Тёмная тема:** Удобный интерфейс, приятный для глаз музыканта.
